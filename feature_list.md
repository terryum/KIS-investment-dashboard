# Feature List: KIS 투자관리 대시보드 v2

## 우선순위 정의

| 등급 | 의미 | 목표 |
|------|------|------|
| **P0** | MVP 필수 | 없으면 대시보드로서 기능하지 않음 |
| **P1** | 1차 릴리스 | 핵심 사용 경험 완성 |
| **P2** | 점진적 추가 | 고도화 및 편의 기능 |

---

## 1. 대시보드

- [P0] 총 자산 금액 표시 (KIS API + 수동자산 합산, 원화 환산)
- [P0] 전일 대비 손익 (금액 + 수익률 %)
- [P0] 자산군별 비중 파이 차트 (주식/채권/현금/기타)
- [P1] 상위 10 보유 종목 바 리스트
- [P1] 포트폴리오 가치 추이 미니 차트 (최근 1개월)
- [P1] 최근 배당/이자 수입 요약 카드
- [P2] 커스텀 위젯 배치 (사용자가 대시보드 구성 편집)

---

## 2. 계좌별 현황

- [P0] 계좌 선택 아코디언 (6+ 계좌 목록)
- [P0] 계좌별 국내주식 보유 목록 (종목명, 수량, 평균단가, 현재가, 평가금, 손익률)
- [P0] 계좌별 해외주식 보유 목록 (USD 기준 + 원화 환산)
- [P0] 계좌별 채권 보유 목록 (만기일, 이자율, 매입금, 평가금)
- [P0] 계좌별 현금 잔고 (원화 + 외화)
- [P0] 수동 자산 추가/편집/삭제 (부동산, IRP, 예금, 대출, ELS/DLS, 해외채권, RP)
- [P1] 종목별 상세 모달 (분류 태그 편집, 거래 이력 요약)
- [P1] 외화 예수금 상세 표시 (USD, CNY 등 통화별)
- [P1] 과거 날짜 기준 조회 (타임머신 - 스냅샷 기반)
- [P2] 수동 자산 변동 이력 차트

---

## 3. 자산 배분

- [P0] 자산군별 배분 (주식/채권/현금/금·원자재) - 도넛 차트 + 테이블
- [P0] 국가별 배분 (한국/미국/중국/인도/유럽/아시아/이머징/글로벌)
- [P0] 통화별 배분 (KRW/USD/CNY/EUR/기타)
- [P1] 주식 구조 분석 (Broad ETF / Factor / Sector / Theme / 개별주)
- [P1] 채권 구조 분석 (만기 래더, HTM vs MTM)
- [P1] 차트 클릭 → 구성종목 팝업 (종목별 비중, 금액)
- [P1] 각 섹션별 AI 인사이트 코멘트 (GPT-4o, 진단→제안→취약점)
- [P2] 타임머신 (과거 날짜 기준 배분 비교)
- [P2] 목표 배분 설정 및 현재 vs 목표 갭 분석
- [P2] 리밸런싱 시뮬레이션 (매수/매도 추천 금액)

---

## 4. 수익률 분석

### 4-1. 수익률 섹션
- [P0] 미실현 손익 현황 (전체 합계 + 종목별)
- [P0] 일별 스냅샷 자동 저장 (Vercel Cron, 21:05 KST)
- [P1] 시계열 수익률 차트 (1W/1M/3M/6M/YTD/1Y/ALL)
- [P1] 종목별 수익률 랭킹 (상위/하위 정렬)
- [P1] 기간별 수익률 비교 테이블
- [P2] TWR (시간가중수익률) 계산 (입출금 보정)
- [P2] MWR (금액가중수익률) 계산
- [P2] 벤치마크 비교 차트 (KOSPI, S&P500 vs 내 포트폴리오)

### 4-2. 인컴 섹션
- [P1] 배당금/분배금/채권이자 내역 테이블
- [P1] 연도별 인컴 합계 (YTD, 전년, 전전년)
- [P1] 월평균 인컴 금액
- [P1] 인컴 유형별 차트 (배당 vs 분배금 vs 이자)
- [P1] KIS API 기반 인컴 자동 감지 (Cron, 08:00 KST)
  - 국내: `CTRGA011R` (기간 권리현황) + `HHKDB669102C0` (KSD 배당일정)
  - 해외: `CTRGT011R` (기간 권리내역 - 배당/유무상 권리 조회)
- [P2] 종목별 배당 이력 및 배당률 추적
- [P2] 인컴 캘린더 (월별 수입 예측)

## 5. 마켓 인사이트 (신규)

- [P2] YouTube/블로그 콘텐츠 요약 (AI)

- [P2] 매수/매도 타이밍 AI 의견
- [P2] 섹터/테마별 시장 동향 대시보드

## 6. 투자 저널 (신규)

- [P1] 매매 기록 작성 (종목, 사유, 진입가, 목표가, 손절가)
- [P1] 보유 종목 메모/분석 작성
- [P1] 저널 목록 조회 (최신순, 종목별 필터)

- [P2] 과거 판단 vs 실제 결과 비교 분석
- [P2] 태그 기반 검색 및 필터

## 7. 공통 인프라

### 보안 (공개 레포 전제)
- [P0] `.env.example` 템플릿 제공 (플레이스홀더 값만, 실제 키 절대 미포함)
- [P0] `.gitignore` 엄격 설정 (.env*, .mcp.json, .vercel/ 등)
- [P0] gitleaks pre-commit hook (계좌번호/API키/대형금액 패턴 탐지)
- [P0] API 키/계좌번호/자산 데이터 하드코딩 금지 (lint 규칙)
- [P0] 모든 KIS API 호출은 서버사이드 전용 (클라이언트에서 직접 호출 금지)
- [P0] Supabase RLS: anon key로 민감 테이블 접근 차단
- [P0] 에러 응답 sanitization (계좌번호/금액 마스킹)
- [P0] 로깅에서 민감 데이터 제외
- [P0] CORS 제한 (허용 origin만)

### 인증
- [P0] 4자리 PIN 인증 (해시 저장, 세션 유지)
- [P0] 모든 API Route에 PIN 인증 미들웨어
- [P0] Cron Job은 CRON_SECRET 헤더 검증

### 테스트 데이터
- [P0] Mock 데이터 전략 (더미 계좌번호, 더미 종목, 더미 금액)
- [P0] 실제 KIS API 응답을 fixture로 커밋하지 않음

### KIS API 연동
- [P0] KIS API 클라이언트 (토큰 관리, 재시도, 에러 핸들링)
- [P0] 다중 계좌 매니저 (6+ 계좌 순회, rate limit 500ms 쓰로틀)
- [P0] 국내주식 조회 API
- [P0] 해외주식 조회 API
- [P0] 채권 조회 API (페이지네이션, 20건/페이지)
- [P0] 펀드 조회 API (연금계좌 포함)

### 데이터 계층
- [P0] Supabase 클라이언트 (서버사이드)
- [P0] DB 마이그레이션 (v1 스키마 기반 + 확장)
- [P0] 환율 API 연동 (한국수출입은행, 주말/공휴일 fallback)
- [P0] IndexedDB 캐시 레이어
- [P0] React Query 프로바이더 (2분 staleTime, 30분 gcTime)

### 레이아웃
- [P0] PC 사이드바 네비게이션 (6개 메뉴)
- [P0] 모바일 하단 탭 바 (반응형 전환)
- [P0] 앱 헤더 (날짜, 새로고침 버튼)

### 배포 및 운영
- [P0] Vercel 배포 설정
- [P0] 환경변수 관리 (Vercel env)
- [P1] Vercel Cron 설정 (일별 스냅샷 + 인컴 감지)

## 구현 순서 요약

```
Phase 0 (P0): 보안 기반 (gitleaks, .gitignore, .env.example)
Phase 1 (P0): 인프라 + 대시보드 + 계좌별 현황
Phase 2 (P0-P1): 자산 배분 + 수익률 분석
Phase 3 (P1): 마켓 인사이트 + 투자 저널
Phase 4 (P2): 고도화 (벤치마크, 리밸런싱, AI 강화, 캘린더)
```
