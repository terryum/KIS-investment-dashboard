# Market Sources — 마켓 인사이트 & 소스 관리

## 소스 큐레이션

### 초기 추천 기준
다양한 관점의 균형을 유지한다:
- 상승론/성장: 2+ 소스
- 하락론/가치: 2+ 소스
- 채권/매크로: 2+ 소스
- 글로벌: 2+ 소스
- 국내: 2+ 소스
- 총 10-15개 소스로 시작

### 소스 유형
YouTube, 블로그, 뉴스레터, 애널리스트 리포트, 팟캐스트, 트위터/X

### 월별 갱신
에이전트가 매월 소스 다양성을 점검하고 추가/교체 제안.

## KIS API 활용

| API | TR_ID | 용도 |
|-----|-------|------|
| 투자의견 | FHKST663300C0 | 증권사 목표가/투자의견 |
| 추정실적 | HHKST668300C0 | 컨센서스 EPS/매출 |
| 배당률순위 | HHKDB13470100 | 고배당 종목 발굴 |
| 뉴스 | news-title API | 국내/해외 시장 뉴스 |
